Pipes

As vezes os dados não estão formatados, nessa hora que Pipes são úteis, pois eles transformam dados que sofrem binding antes destes serem exibidos, para que estem fiquem mais amigáveis ao usuário.

Angular fornece alguns Pipes próprios (built-in) para formatar valores, inclusive em property binding

O código abaixo mostra a utilização de 2 pipes, currency (formatar em moeda)
{{"{\{20 | currency }\}"}}
//{{20 | currency}}

 O código abaixo mostra a utilização do pipe currency, mas enviando os seguites parâmetros:
{{"{\{3.9 | currency:'BRL':true:'1.2.2'}\}"}}
//{{3.9 | currency:'BRL':true:'1.2-2'}}
  1. O código da moeda, no caso, 'BRL' (reais do Brasil)
  2. Valor booleano para mostrar ou não o símbolo da moeda
  3. '1.2-2': 1 representa o mínimo de números inteiros, 2-2 representa o mínimo e máximo de números nas casas decimais

Alguns pipes para valores

  • date
  • number
  • decimal
  • percent
  • currency
  • uppercase
  • lowercase

Alguns pipes para objetos

json
- mostra o conteúdo de um objeto como uma string JSON, útil para debugar aplicações
slice
- seleciona um número específico de itens de uma lista
